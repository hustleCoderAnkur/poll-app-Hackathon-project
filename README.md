# Poll App

A full-stack real-time polling application where users can create polls, vote instantly, and view live poll results.

## Repositories

### Frontend
https://github.com/hustleCoderAnkur/poll-app-frontend

### Backend
https://github.com/hustleCoderAnkur/poll-app-backend

---

# Features

- Create custom polls
- Vote on polls
- Real-time poll updates
- Responsive UI
- REST API integration
- Full-stack architecture

---

# Tech Stack

## Frontend
- React.js
- Tailwind CSS
- Axios

## Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
